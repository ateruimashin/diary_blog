そろそろ家のPCが3年目に突入したのでパーツを変えなきゃなあと思っているところです。

最近はPCゲームをあまりやらなくなってしまいましたが、やはりPCのスペックはある程度持ってないと心の余裕がない。

そいうわけで、PCパーツの更新を検討してみました。

<!-- more -->

# 今の構成

今の構成は以下の通り。

| パーツ | 型番・スペック |
|--------|--------------|
| CPU | Intel i5-4690K |
| マザーボード | ASUS ROG（型番不明） |
| GPU | GTX 970 |
| メモリ | 8GB + 4GB（メーカー不明、DDR3） |
| 電源 | Cooler Master V750 |
| ストレージ | SSD 128GB / SSD 256GB / HDD 4TB / HDD 500GB（各種メーカー不明） |

メモリの主流もDDR3からDDR4に変わってますし、Intel CPUのソケットもLGA1151になってますし、ほとんど取り替えです。

電源だけはまだ使えるのかなあ？と思いますが、24時間365日つけっぱなしのPCの電源で2年経ってたら・・・どうなんだろう?

とりあえず、CPU, GPU, マザーボード, メモリは交換することにします。

# CPU

CPUはIntel,AMDのどっちかですが、最近ZenでAMDが元気なのでAMDも気になっています。

友人の一人がAMDerなので、おそらく相談したらAMD勧めてくるだろうなあ・・・。

とりあえず候補なら以下。

| メーカー | 型番 | 価格 |
|----------|------|-----:|
| Intel | Core i7-8700K | 40,460円 |
| AMD | Ryzen 7 2700X | 40,645円 |

そういやリテールクーラーついてこないんでしたっけ。

CPUクーラーなら、Hyper H411R RR-H411-20PW-R1でいいかなあって感じです。

というか3000円で普通に買えるから、パーツ買うときに買い忘れないように注意しなきゃ。

価格.com眺めてたら、簡易水冷が人気高いみたいなんですが、簡易水冷ってそんなに冷えない、うるさい、空冷でいいって言うイメージなんですが、今は違うのかなあ。

本格水冷は・・・私の怠惰さでやると絶対やばいので・・・。

# マザーボード

特にほしい機能がない・・・ので、適当に候補を出してしまいましょう。

| 型番 | ソケット | ストレージ | 備考 | 価格 |
|------|----------|-----------|------|-----:|
| PRIME H370-A | LGA1151 | M.2×2 / SATA3×6 | Optane Memory Ready | 12,589円 |
| TUF Z370-PLUS GAMING | LGA1151 | M.2×2 / SATA3×6 | Optane Memory Ready、耐久性高め（プレスリリース） | 16,476円 |
| ROG CROSSHAIR VI HERO | AM4 | M.2×2 / SATA3×8 | SATA3×8は良い | 21,578円 |

**ASUSばっかじゃねえか！！！**

ASUSって特に外れないのと、私が好きなブランドなんで・・・はい。

TUF Z370-PLUS GAMINGは耐久性が高め(ってプレスリリースで言ってる)ので私の用途だと良さそうです。

ぶっちゃけチップセットごとの違いとか知らないので、Z370買っとけばいいんでしょ！的なところはあります。

あと無駄に光らせる機能がないのでいいです。

ROG系は最近無駄に光らせたがるところありますし。

ROG CROSSHAIR VI HEROはSATA3*8なのがとてもいいです。

ストレージって勝手に増えますからね、ほんと。

多分買うならこのどっちかな気がします。

H370-Aはいいマザーボードですけど、なんかなーみたいな感じがあります。

# GPU

買うときには夏以降になるので、GTX11xxシリーズあたりになりそうです。

ご祝儀相場で買いたくねえなあ・・・って感じです。

アスク税高すぎだろ、まじで。

| 型番 | 価格 |
|------|-----:|
|GTX1170|登場当初約8万円(アメリカ$499)|

・・・Radeonはさっぱりわからない・・・ので・・・申し訳ない。

GTX1180とか買えればいいんですけど、そんなにお金ないので。

GTX1170でGTX1180tiと同等の性能らしいので、2年ぐらい現役で行けるでしょ。

# メモリ

DDR4メモリ大分値下がりしましたね。

いい傾向です。

流石にドル円が80円のときのやべえ価格までは下がらないので、2666で適当に選べばいいかなあって感じです。

DDR4-2666 16GBで約2万円前後なんですが、G.SkillがDDR4-3600 16GBを22800円で売ってる・・・。

2666と3600ってそんなに変わるんですかね？

MSIに比較ありましたが、fps10ぐらい差が出る・・・？

本当かなあ？ぶっちゃけ何でもいい気がする。

# ストレージ

気がついたらHDD6TBが13000円で買えるようになったんですね。

ということで、今の4TBを6TBに交換です。

むしろ、500GBの何に使ってるかわからないHDDと交換のほうがいい気がしてきた。

4TB近いデータを移したくないよ・・・、一体何時間かかるんだろう？

システムドライブは256GBSSDで問題ないので、問題はDドライブにしているSSD128GBですね。

128GBって使い道ないなあって去年ぐらいから思っています。

でかいゲーム入れたらもう何も入らない。

ここは512GBに交換したいところ。

もう512GBも15000円程度で買えるようになってますね。

数年前は2倍以上だったのに・・・。

また、やはりM.2は入れておきたいところ。

IntelのSSD 760p SSDPEKKW512G8XTでいいでしょう。

960EVOのほうがコスパいいんですけど、Samsung製っていまいち触手が伸びない・・・。

M.2は結構発熱するらしいので、ヒートシンクがいるそうです。

マザボについてくるやつあったけど、上の候補のやつにあったっけ？

まあ買えばいいんですけどね。

ということで、ストレージは以下の通り。

| 種別 | 型番 | 価格 |
|------|------|-----:|
| HDD | WD60EZRZ-RT（6TB SATA600 5400rpm） | 13,330円 |
| SSD | Crucial MX500 CT500MX500SSD1/JP | 14,864円 |
| M.2 SSD | Intel 760p SSDPEKKW512G8XT | 24,796円 |

# まとめ

2年前と比べてストレージが大分値下がりしていたので驚きました。

また、そろそろComputexなので、CPUあたりは9700Kとか出てくるんですかね？

グラボの方は性能の向上が凄まじいので、欲しくなったときに買えばいいや的なスタンスです。

とりあえず、予算としては25万用意しとけばいい感じですね。

ロードバイクぐらいの予算をもう一回用意しなきゃならんのか・・・。

あとはサングラス・・・極度の近視の乱視の私には選択肢がなさすぎて辛い。

あとレーミクさんのレージャーほしい。

バニラも出てきそうだし、二人の立ってるやつもめっちゃほしい。

あーもう、**無限に金がほしい!!!!**
