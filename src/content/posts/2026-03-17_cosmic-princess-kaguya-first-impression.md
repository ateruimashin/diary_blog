話題になっている「超かぐや姫！」を映画館で見てきました。

本当に見に行って良かったです。

感想にはネタバレが含まれるので注意してください。

<!-- more -->

# ボカロオタクとして

作品自体は知っていたものの、それはボカロオタクとしての延長線上でした。

そのため、作品自体にはそれほど興味がなかったです。

むしろ、「ボカロ黎明期の神曲をRemixしたりカバーしているけど、原曲の方が良いに決まっているだろ」という逆張りオタクな態度でした。

作品を鑑賞した後には、一転して「この作品があるならRemixもカバーも納得だ」と思うようになりました。

ちゃんと作品の文脈に合致していた。

メルトの「好きなの」という部分が必要な理由が魂で理解できた。

ワールドイズマインの一枚絵を再現していたのも良かったし。

また、rayがEDで流れたときは泣きました。

rayは初音ミクとコラボした曲もあり、あのときは両方から「なんでそんなコラボをするんだ」と非難されている部分があった覚えがあります。

私は「なんでボカロをそんなメジャーなところに出すんだ、ボカロというだけで叩かれるだろ」と思っていたボカロオタクでした。

しかし、MVを見てその考えは間違えだったと恥じた思い出があります。

その曲がこの作品の最後を飾るのですから、泣かないわけがありません。

# ただのオタクとして

ほんの一部分を切り出すと他作品のパロディや懐かしい要素があったりして面白かったです。

初めてエントリープラグにLCLを流し込まれたシンジ君と同じことをしているカグヤだったり。

他にはきららジャンプしていたり。

アホ毛表現も懐かしかったです。

# 百合

彩葉とカグヤの握手が尊かったです。

全体的に尊いですね。

# かぐや姫と超かぐや姫

古典の授業で竹取物語は勉強しました。

「今は昔、竹取の翁というものありけり」から始まる数ページを暗唱させられた覚えがあります。

作品中でも古典の授業をしているシーンが幾つかありました。

求婚されて無理難題を出して事実上振るってのはカットされていた気がしますが、求婚自体はされていたでいいんでしょうか。

ただ、最後の最後のシーンの山に登ってたけのこを埋めるのは、原典通りっぽそう。

原典では不老不死の薬を燃やしたので不死山(富士山)になったとなっていましたが、あの山はどこなんでしょう。

作品を鑑賞中、古典の授業をちゃんと受けていて良かったな～と思っていました。

# 自己表現の手段を持たぬ者

とてもいい作品でした。

でも、家に帰って落ち込みました。

昔から自己表現の手段を持たない者はどうしたらいいんだ、というのを考えています。

この作品も、自己表現の手段を持った者が中心です。

私の考えている自己表現の手段というのが自分の思っているものを外部に公開できる手段です。

倫理の"昇華"ができる手段かもしれません。

以前、このことを友人にも相談したことがあります。

その時の返答が、「え、ブログ書いているならそれなんじゃないの」でした。

確かに文章を書き、それを公開しているなら自己表現ではありそうです。

ただ、私にとってブログに書くということは記録であり、自己表現ではないと思っています。

記憶からなくなっていくから、ブログ記事として記録しておく。

記録は客観的である必要がある。

そのため、感情すらも客観的に評価して記録する必要がある。

つまり、私が思う自己表現の手段というのは、一時の感情を爆発させる手段と言い換えられると思います。

一時の感情を別の方面に爆発する方法が無いと、その感情の扱いがわからなくなります。

結局、感情を忘れるか、忘れたことにするしかなくなります。

この手段は誰もが持っているわけでは無く、努力をした結果手に入るものだと思っています。

この部分は私のコンプレックスの根深い部分なので、普通の人はそうでないのかもしれないです。

感想というか、私の根深い劣等感が爆発した話でした。

# 舞台装置

初っ端、彩葉の設定を説明されたとき、「そんなことが可能か??」となってしまいました。

ああこれ、作品を成り立たせるための舞台装置感が強いなと。

勉強ができて、バイトもやっていて、「久しぶりに6時間寝れる」ぐらいの睡眠時間で、美人で、友人もいて、ゲームもできて、生活費と学費を自分で稼いでいて。

物語が進むにつれて、その感想は間違いだったと気が付かされました。

特に、ライブのために模試を休むの、心情の変化が見られてすごく良かったです。

人に頼れないから人に頼るようになるところや、がむしゃらに頑張るところから一旦寄り道をするようになるところなど、完璧超人が柔軟になっていく様子はとても良いものでした。

# 描画の仕方

現実とツクヨミで描き方が全然違うなって思いました。

現実のキャラの線の書き方と、ツクヨミでのキャラの線の書き方だと、ツクヨミのほうがしっかり書かれているように感じました。

見た後にtwitterで「現実ではキャラの耳はそんなにちゃんと書かれていないけど、ツクヨミだとキャラの耳がちゃんと描かれている」って言うのを見て、たしかにそうだったなと。

この描画の緩急が面白かったです。

# 親との関係

ここ10年ぐらいアニメをほとんど見ていなくて、最近見たアニメがミルキーサブウェイという感じなんですけど、最近の親と子の関係の共通項って"良くはないしどっちかといえば悪い方"なんでしょうか。

この作品は、「それ扶養義務果たしてなくない？」ってなるレベルの母親と子の関係になっています。

どうやらカグヤが彩葉に会うためにフラグを立てる必要があり、そのフラグの一つに彩葉を一人暮らしさせるというのがあるという考察は見たのですが、それにしてもココまで親子関係が悪化させるか。

大学の時の同期にも、学費は払ってやるけど家賃と生活費は自分で稼げと言われて一人暮らししている人もいました。

私も中学～大学までは母親との関係はお世辞にも良いというものではありませんでした。

中学、高校の時期は親と子の関係が拗れやすいとは思うのですが、これは今の中高生も変わらないのでしょうか。

# 不動産

アニメのストーリーに文句言ったらいけない部分ではありますが...。

まず、バイトして学費と生活費を稼いでいる状態から保証人を付けたとして家賃が月30万円ぐらいの部屋を借りれると思えないんです。

そもそも、高校生が容易に高い不動産を手に入れられたってのがすごい納得がいっていないんですよね。

私は自分の人生の価値を銀行に計られて、妥協した結果に不動産を手に入れたのですが、普通に不動産を手に入れられると嫉妬で狂いそうになりました。

本当に嫉妬です。

あと、カグヤが月へ帰ったときに振り込まれた大金は贈与なのか雑収入なのか気になりました。

本当に蛇足ですみません。

# おわりに

見てよかったです。

いつもみたいに先延ばしにして、見る機会を失わなくて良かったです。

あと2回は見たいですね。
